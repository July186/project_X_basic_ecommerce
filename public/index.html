Solusi Masalah Deployment Vercel

Memperbaiki masalah path, asset, dan routing pada website CartSale

Masalah yang Ditemukan

Path Asset Tidak Konsisten

Beberapa file menggunakan path absolut (/Asset/...) dan lainnya menggunakan path relatif (../Asset/...).

Halaman Detail Produk Tidak Tampil

Kemungkinan masalah dengan routing parameter ID atau path yang salah.

Link Register/Login Tidak Berfungsi

Ada perbedaan path antara file JS dan HTML untuk halaman register/login.

Asset Tidak Konsisten Antar Halaman

Beberapa asset tampil di halaman home tapi tidak di halaman cart.

Solusi yang Direkomendasikan

Gunakan Path Relatif yang Konsisten

Pastikan semua path asset menggunakan format yang sama. Disarankan menggunakan path relatif terhadap root.

// Contoh path yang benar
<img src="/assets/images/logo.png" alt="Logo">
<script src="/js/script.js"></script>

Perbaiki Routing Detail Produk

Pastikan parameter ID terbaca dengan benar dan endpoint API dapat diakses.

// Pada DetailProduk.js
const urlParams = new URLSearchParams(window.location.search);
const productId = urlParams.get("id");

if (productId) {
  detailcontainer(productId);
} else {
  console.error("Product ID not found in URL");
}

Perbaiki Link antar Halaman

Pastikan link register/login mengarah ke path yang benar dan konsisten.

// Contoh link yang konsisten
<a href="/login.html">Login</a>
<a href="/register.html">Register</a>

Struktur Folder yang Jelas

Buat struktur folder yang terorganisir untuk memudahkan management asset.

/
├── index.html
├── home.html
├── cart.html
├── detail-produk.html
├── checkout.html
├── login.html
├── register.html
├── /assets
│ ├── /images
│ ├── /icons
│ └── /css
└── /js

Implementasi pada Vercel

Penting untuk Deployment di Vercel:

1. Pastikan Anda memiliki file vercel.json untuk konfigurasi routing

2. Gunakan path absolut dengan awalan "/" untuk semua asset

3. Pastikan semua file HTML memiliki struktur yang konsisten

Contoh vercel.json:

{
  "routes": [
    { "handle": "filesystem" },
    { "src": "/(.*)", "dest": "/index.html" }
  ],
  "rewrites": [
    { "source": "/home", "destination": "/home.html" },
    { "source": "/cart", "destination": "/cart.html" },
    { "source": "/product/:id", "destination": "/detail-produk.html" }
  ]
}

Contoh Perbaikan Path pada Home.js:

// Ganti path relatif dengan absolut
// Dari:
<a href="detail-produk.html?id=${post.id}">
<img src="${post.image}" alt="${post.title}" class="w-full h-44 object-contain mb-4 mt-4" />

// Menjadi:
<a href="/detail-produk.html?id=${post.id}">
<img src="${post.image}" alt="${post.title}" class="w-full h-44 object-contain mb-4 mt-4" />
